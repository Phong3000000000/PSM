# Portal Contact Update Module

## Description
Allow candidates to upload personal documents via Odoo portal.

## Features
- Portal form for document upload
- Email invitations with custom template
- Progress tracking in contact form
- Auto-sync documents to Odoo backend

## Required Documents
1. Passport Photo (Ảnh thẻ)
2. ID Card - Front & Back (CCCD/CMND)
3. Household Registration (Hộ khẩu bản sao)
4. Judicial Record (Lý lịch tư pháp)
5. Professional Certificate (Bằng cấp chuyên môn)

## Installation
1. Copy module to `addons` or `custom_modules`
2. Restart Odoo
3. Apps → Update Apps List
4. Search "Portal Document Upload"
5. Install

1. Create Contact with email
2. Action → Grant Portal Access
3. User receives email → uploads documents
