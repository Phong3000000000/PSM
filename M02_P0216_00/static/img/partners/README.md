# Urbox Partner Images

This folder contains logo images for Urbox partners.

## Instructions

To add partner logos:

1. Place logo images in this folder (recommended size: 256x256 px)
2. Supported formats: PNG, JPG, SVG
3. Recommended naming: `partner_code.png` (e.g., `grab.png`, `shopee.png`)

## Demo Partners

The following partners are included in demo data:

- **CGV Cinemas** - cgv.png
- **KFC Vietnam** - kfc.png
- **Lotte Cinema** - lotte.png

## Usage

You can upload partner logos through the Odoo interface:
1. Go to **Voucher → Đối tác Urbox**
2. Create or edit a partner
3. Click on the logo placeholder to upload an image

Logos are stored in the database as Binary fields for better portability and backup.
